Project Objective

To develop a comprehensive credit card weekly dashboard that provides real-time insights into key performance metrics and trends, enabling stakeholders to monitor and analyze credit card operations effectively.

DAX Queries

Create Age Group

                AgeGroup = SWITCH(
                TRUE(),
                'public cust_detail'[customer_age] < 30, "20-30",
                'public cust_detail'[customer_age] >= 30 && 'public cust_detail'[customer_age] < 40, "30-40",
                'public cust_detail'[customer_age] >= 40 && 'public cust_detail'[customer_age] < 50, "40-50",
                'public cust_detail'[customer_age] >= 50 && 'public cust_detail'[customer_age] < 60, "50-60",
                'public cust_detail'[customer_age] >= 60, "60+",
                "unknown"
            )
Create Income Group

                IncomeGroup = SWITCH(
                TRUE(),
                'public cust_detail'[income] < 35000, "Low",
                'public cust_detail'[income] >= 35000 && 'public cust_detail'[income] <70000, "Med",
                'public cust_detail'[income] >70000, "High",
                "unknown"
            )
Calculate Revenue

                Revanue = 'public cc_detail'[annual_fees] + 
                      'public cc_detail'[total_trans_amt] + 
                      'public cc_detail'[interest_earned]
Current and Previous Week Revenue

                Week_num2 = WEEKNUM('public cc_detail'[week_start_date])
            
            Current_week_Revanue = CALCULATE(
                SUM('public cc_detail'[Revanue]),
                FILTER(
                    ALL('public cc_detail'),
                    'public cc_detail'[Week_num2] = MAX('public cc_detail'[Week_num2])
            ))
            
            Previous_week_Revanue = CALCULATE(
                SUM('public cc_detail'[Revanue]),
                FILTER(
                    ALL('public cc_detail'),
                    'public cc_detail'[Week_num2] = MAX('public cc_detail'[Week_num2])-1
            ))

How to Run the Project

  1. Set up the PostgreSQL database and import the data using the provided SQL scripts.
  2. Open the Power BI dashboard file and connect it to the PostgreSQL database.
  3. To test real-time functionality, add data to cust_add.csv and cc_add.csv and refresh the Power BI dashboard.
