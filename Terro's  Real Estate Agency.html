Project Objective

The project aims to identify the most influential factors affecting house prices in a particular locality through exploratory data analysis and regression modeling.

Finding out the most relevant features for pricing of a house

Terro’s real-estate is an agency that estimates the pricing of houses in a certain locality. The pricing is concluded based on different features / factors of a property. This also helps them in identifying the business value of a property. To do this activity the company employs an “Auditor”, who studies various geographic features of a property like pollution level (NOX), crime rate, education facilities (pupil to teacher ratio), connectivity (distance from highway), etc. This helps in determining the price of a property.

Task

Your job, as an auditor, is to analyze the magnitude of each variable to which it can affect the price of a house in a particular locality. To do the analysis, you are expected to solve these questions:

  1. Generate the summary statistics for each variable in the table. (Use Data analysis tool pack). Write down your observation.
  2. Plot a histogram of the Avg_Price variable. What do you infer?
  3. Compute the covariance matrix. Share your observations.
  4. Create a correlation matrix of all the variables (Use Data analysis tool pack):
    • a) Which are the top 3 positively correlated pairs?
    • b) Which are the top 3 negatively correlated pairs?

Key Insights

1. Summary Statistics
  • The dataset contains 506 records with variables such as crime rate, age, tax, NOX level, pupil-teacher ratio, etc.
  • Variables like Distance, Age, and Tax showed significant variance.
  • Most houses are located far from highways (Distance mode = 24) and have an age of 100 years (Age mode = 100).
  • The dataset is highly skewed, indicating non-normal distributions.
2. Histogram (Avg_Price)
  • Most houses are priced between $21,000 to $25,000.
  • Very few houses fall in the $37,000–$41,000 and $45,000–$49,000 range.
3. Covariance Matrix Observations
  • High covariance between Tax and other variables (excluding crime rate) implies it strongly affects other house characteristics.
  • This indicates Tax is an important influencing feature in price variability.
4. Correlation Matrix Observations

Top 3 Positively Correlated Pairs:

  • Distance – Tax
  • NOX – Age
  • NOX – Indus

Top 3 Negatively Correlated Pairs:

  • LSTAT – Avg_Room
  • Avg_Price – PTRATIO
  • Avg_Price – LSTAT
5. Regression Modeling

Model 1: Simple Regression (Avg_Price ~ LSTAT)

  • R² = 0.54 → 54% of price variance explained by LSTAT.
  • Coefficient = -0.95 → Higher LSTAT (lower income) leads to lower Avg_Price.
  • Statistically significant (p-value < 0.05).

Model 2: Multiple Regression (Avg_Price ~ LSTAT + Avg_Room)

  • R² = 0.64 → 63% variance explained.
  • Equation: Avg_Price = -1.358 + 5.09 * Avg_Room - 0.642 * LSTAT
  • Prediction for 7 rooms and LSTAT = 20: $21,440 vs quoted $30,000 → Overcharging detected.
6. Full Model with All Variables
  • Adjusted R² = 0.69 → 69% variance explained.
  • Significant negative coefficients: NOX, TAX, PTRATIO, LSTAT.
  • Crime rate found to be statistically insignificant.
7. Optimized Model with Only Significant Features
  • Variables used: DISTANCE, TAX, PTRATIO, AVG_ROOM, LSTAT.
  • Adjusted R² = 0.68 → Slightly lower than full model but more efficient.
  • All variables statistically significant.

Conclusion

Features like NOX, PTRATIO, TAX, and LSTAT negatively impact housing prices. Higher pollution, poor student-teacher ratios, and higher taxes reduce value. Avg_Room and Distance positively affect price, showing that buyers prefer more spacious homes away from dense urban pollution. Crime rate is not a significant predictor in this dataset. A simplified model with fewer variables still predicts price with around 68% accuracy, making it useful for practical property evaluations.
